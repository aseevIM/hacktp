# Автоматизированная система обработки почты с расширением(!!!)
# Начнем с описания нашего проекта) Мы сделали программу, которая распределяет папку с письмами в папку с категориями, но помимо этого мы сделали два(!!!) очень полезных расширения: наша программа генерирует  # # HTML файл с 
# диаграммами, которые показывают сколько пришло писем из разных категорий и в количественном, и в процентном соотношении! Помимо этого мы выводим приложение, в котором так же показывается статистика, и #пользователь сам сможет выбирать, каким расширением ему воспользоваться! Классификация у нас идет по 5-ти категориям + Other для необработаннных писем.

# Сам проект работает довольно примитивно:
# 1) Сначала мы запускаем bash скрипт, который проверяет все ли на месте: папки, приложение python, чтобы впоследствии мы могли запустить нашу прогу :)
# 2) Ну и запускается сама прога с помощью скрипта!
# 3) Потом /inbox считывается email_importer, чтобы в дальнейшем классифицировать данные из писем!
# 4) Затем запускается classifier, который классифицирует письма!(спам, важное и тд)
# 5) Дальше в ход вступает distributor, который распределяет письма по соответствующим папкам! вместе с ним отчет и лог!
# 6) И в завершение наши расширения! visualizer делает html с диаграммами, а арр собирает данные(опять же, все по выбору пользователя)
